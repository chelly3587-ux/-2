1 (예제 6.1-10) 삼각함수의 직교성
* 1단계 (What): 삼각함수의 적분 공식 $\cos A \cos B = \frac{1}{2}[\cos(A+B) + \cos(A-B)]$ 를 적용한다. $\alpha, \beta$ 가 정수일 때, $\alpha \neq \beta$ 이면 $I = \int_{-\pi}^{\pi} \frac{1}{2}[\cos(\alpha+\beta)x + \cos(\alpha-\beta)x] dx = 0$ 이다.  $\alpha = \beta \neq 0$ 이면 $I = \int_{-\pi}^{\pi} \cos^2 \alpha x dx = \pi$ 이며, $\alpha = \beta = 0$ 이면 $I = 2\pi$ 이다.
*  2단계 (So What): 이 결과는 서로 다른 주기를 가진 코사인 함수들이 특정 구간에서 서로 '직교'함을 수학적으로 보여줌. 이는 복잡한 신호를 단순한 삼각함수의 합으로 분해하는 푸리에 해석의 핵심적인 기초가 됨

 
2 (예제 6.2-2) 치환적분법의 기초
* 1단계 (What): $u = 4-x^2$ 으로 치환하면 $du = -2x dx$ 가 된다. 따라서 $I = \int \frac{-1}{\sqrt{u}} du = -2\sqrt{u} + C$ 이며, 원래 변수로 되돌리면 $I = -2\sqrt{4-x^2} + C$ 이다. 치환적분법을 통해 복잡한 무리함수 형태를 기본 적분 꼴로 변환하였다.
* 2단계 (So What): 피적분 함수 내부의 미분값이 외부에 곱해져 있는 구조를 파악하여 계산을 단순화함. 이는 변화율의 연쇄적 관계를 역으로 추적하여 '전체 양'을 찾아내는 과정을 의미함


3 (연습문제 6.2-5) 유리함수와 역삼각함수
* 1단계 (What): 분모를 표준형으로 고치면 $x^2+2x+5 = (x+1)^2+2^2$ 이다. $\int \frac{1}{x^2+a^2} dx = \frac{1}{a}\tan^{-1}(\frac{x}{a}) + C$ 공식을 적용하기 위해 $u=x+1$ 로 치환하면, $I = \frac{1}{2}\tan^{-1}(\frac{x+1}{2}) + C$ 를 얻는다.
* 2단계 (So What): 실수 범위에서 해가 없는 2차식 분모를 가진 유리함수의 적분 결과가 역삼각함수로 나타남을 보여준다. 이는 대수적인 유리함수 시스템이 기하학적인 회전이나 각도의 개념과 연결될 수 있음을 보여줌


4 (연습문제 6.2-6) 역쌍곡선함수 치환
* 1단계 (What): $x = \sec \theta$ 또는 $x = \cosh t$ 로 치환할 수 있다. $x = \cosh t$ ($t > 0$ )로 치환하면 $dx = \sinh t dt$ 이고 $\sqrt{\cosh^2 t - 1} = \sinh t$ 이므로, $I = \int 1 dt = t + C = \cosh^{-1} x + C$ 이다. 로그 형태로 표현하면 $\ln(x + \sqrt{x^2-1}) + C$ 이다.
* 2단계 (So What): 무리함수 형태의 적분 경로가 쌍곡선 함수의 역함수 관계를 따름을 의미함


5 (연습문제 6.3-2) 부분분수 분해
* 1단계 (What): 분모를 인수분해하면 $(x+6)(x-1)$ 이다. $\frac{x+4}{(x+6)(x-1)} = \frac{A}{x+6} + \frac{B}{x-1}$ 로 두면 $A=\frac{2}{7}, B=\frac{5}{7}$ 이다. 각 항을 적분하면 $I = \frac{2}{7}\ln|x+6| + \frac{5}{7}\ln|x-1| + [cite_start]C$ 이다.
* 2단계 (So What): 복잡한 하나의 유리함수 시스템을 독립적인 두 개의 단순한 로그 시스템의 합으로 분해함. 이는 복잡한 전체 구조를 분할 정복('거대한 문제를 해결하기 쉬운 작은 문제로 나누어 각각을 해결한 뒤, 그 결과를 합쳐 전체 문제를 푸는 알고리즘 설계기법' 라고 함) 의 원리를 보여준다.


6 (예제 6.4-4) 이상적분의 수렴성
* 1단계 (What): $I = \lim_{t\to\infty} \int_1^t x^{-p} dx$ 를 계산한다. $p=1$ 이면 $\ln x$ 가 되어 발산하고, $p \neq 1$이 면 $[\frac{x^{1-p}}{1-p}]_1^t$이다. $1-p < 0$ , 즉 $p > 1$ 일 때만 극한값이 존재하여 수렴한다.
* 2단계 (So What): 함수가 무한히 멀어질 때 0으로 감소하는 속도가 "충분히 빨라야"($p>1$) 전체 면적이 유한해질 수 있음을 의미함. 이는 물리적 시스템에서 에너지나 질량의 총합이 유한하기 위한 "최소한의 감쇠 조건"(중요한듯)을 결정하는 조건을 보여줌


7 (예제 7.1-3) 포물선과 직선의 영역
* 1단계 (What): 두 곡선의 교점 $x^2 = x+2 \Rightarrow (x-2)(x+1)=0$ 에서 $x=-1, 2$ 를 구한다. 구간 $[-1, 2]$ 에서 직선이 위쪽에 있으므로 넓이 $S = \int_{-1}^2 (x+2 - x^2) dx = [\frac{x^2}{2} + 2x - \frac{x^3}{3}]_{-1}^2 = \frac{9}{2}$ 이다.
* 2단계 (So What): 비선형적인 포물선과 선형적인 직선 사이의 폐곡선 면적을 정적분으로 계산함. 두 함수 사이의 차이를 "누적"함으로써 복잡한 경계를 가진 평면의 크기를 정확히 계산할 수 있음을 보여줌


8 (연습문제 7.1-10) 함수의 차에 의한 넓이
* 1단계 (What): 구간 $[0, \pi/2]$ 에서 $\sin x$ 와 $\cos x$ 는 $x=\pi/4$ 에서 만난다. 따라서 $S = \int_0^{\pi/4} (\cos x - \sin x) dx + \int_{\pi/4}^{\pi/2} (\sin x - \cos x) dx$ 로 나누어 계산한다. 계산 결과 $2\sqrt{2}-2$ 가 나온다.
* 2단계 (So What): 두 주기 함수의 "대칭성"과 "상대적 높이 변화"를 고려하여 면적을 구했다. 위아래 관계가 역전되는 지점(교점)을 기준으로 적분 구간을 분리하는 것이 기하학적 해석의 핵심임을 보여준다.
