1 (예제 6.1-10) 삼각함수의 직교성
* 1단계 (What): 삼각함수의 적분 공식 $\cos A \cos B = \frac{1}{2}[\cos(A+B) + \cos(A-B)]$ 를 적용한다. $\alpha, \beta$ 가 정수일 때, $\alpha \neq \beta$ 이면 $I = \int_{-\pi}^{\pi} \frac{1}{2}[\cos(\alpha+\beta)x + \cos(\alpha-\beta)x] dx = 0$ 이다. [cite_start]$\alpha = \beta \neq 0$ 이면 $I = \int_{-\pi}^{\pi} \cos^2 \alpha x dx = \pi$ 이며, $\alpha = \beta = 0$ 이면 $I = 2\pi$ 이다.
*  2단계 (So What): 이 결과는 서로 다른 주기를 가진 코사인 함수들이 특정 구간에서 서로 '직교(Orthogonal)'함을 수학적으로 보여줌. 이는 복잡한 신호를 단순한 삼각함수의 합으로 분해하는 푸리에 해석의 핵심적인 기초가 됨
